NAME
mountnfs – Connect to and mount NFS service

SYNOPSIS
mountnfs nfshost export

DESCRIPTION
The rc(1) script mountnfs invokes nfs(4) to create at /srv/nfs.nfshost a channel to the NFS service on nfshost, then mounts nfshost's export at /n/nfshost.

EXAMPLE
Make available at /n/intranet NFS server intranet's export /home/public:
% mountnfs intranet /home/public

FILES
/lib/ndb/nfshost.^(password group)
Expected UNIX–format password and group files for uid–to–name mapping. See –u in nfs(4).

SOURCE
/rc/bin/mountnfs

SEE ALSO
mount(1), nfs(4), srv(3)
